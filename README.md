# Engineer Project: Web

**Subject**: Application for collecting information about popculture

## How to launch?

-   In client and server copy `.env.example`, rename it to `.env` and fill it with missing data

### With Docker

-   Run `docker compose up`

### or without

#### Server

-   Run `npm i`
-   Run `npm run dev`

#### Client

-   Run `npm i`
-   Run `npm run dev`
